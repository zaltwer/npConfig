【ネコペイント設定用GUI　neConfig　Ver1.0.0.1　2014/04/24】

■説明
・ペイントソフト「ネコペイント」のショートカットキー設定、スクリプト設定を行う外部ツールです。
　ショートカットキー設定ファイル「key.txt」および、スクリプト設定ファイル「list.txt」「user_list.txt」の書き換えを行います。

【重要】
・このツールはネコペイント作者様とは一切無関係です。
　このツールについてのネコペイント作者様への質問、要望はご遠慮ください。

・使用前にはネコペイント設定ファイルのバックアップを取ることをおすすめします。
　このツールの使用によるデータの破損、機器への損害については自己責任でお願いします。

・既存のショートカット設定で複数機能に同じショートカットが割り当てられている場合
　2件目以降の設定は無視されます。

・スクリプト設定ファイルで所定の書式で書かれていない行は削除されます。
　また、#以降のパスの中に無名が含まれる行についても削除されます。

【超重要】
・ころんでも泣かない


■動作環境
・ネコペイント本体(npaint_script.exe)
・.NET Framework 4　以上(client profile可)

■使い方
・圧縮ファイルを解答すると以下のファイルが生成されるので、neConfig.exeを起動して下さい。
　neConfig.exe　(ツール本体)
　neConfig.exe.config　(このツールの設定ファイル。同じフォルダにないと動きません)
　Readme.txt　(このファイル)

・初回起動時にネコペイント本体の場所を聞かれるのでnpaint_script.exeの場所を指定して下さい。
　次回起動時からはここで指定したネコペイントをターゲットにします。
　別のネコペイントを指定したい場合は以下のフォルダを削除して下さい。(「ユーザー名」には使用しているユーザー名が入ります)
　C:\Users\「ユーザー名」\AppData\Local\neConfig

【ショートカット設定タブ】
・なんとなく見た目通りに動くと思います
・左のリストから選んだ機能に右の入力エリアでショートカットを設定して下さい。

【スクリプト設定タブ】
・ネコペイントで使用できるスクリプト機能の設定・およびスクリプトへのショートカット割り当てを行います
・左のツリービューが実際にネコペイントのメニューに表示されるスクリプトになります
　（ツリービューではドラッグでスクリプトの移動が出来ます）
・ツリービューで選択したスクリプトに対し真ん中あたりで設定が出来ます。
・スクリプトファイル名のドロップダウンリストにはネコペイントフォルダ配下の
　\script\menuにある.nasファイルの一覧が表示されます。
・スクリプト名のドロップダウンリストにはスクリプトファイル名で選択したファイル内にある使用可能なスクリプトが表示されます。
・ソースビューにはスクリプトファイル名で選択したファイルの内容が表示されます。
・ショートカット設定タブと同様にツリービューで選択したスクリプトに対し右のエリアでショートカットが設定できます。

【ショートカット保存ボタン】
　ショートカット設定ファイル「key.txt」のみを保存します。

【全設定保存ボタン】
　ショートカット設定ファイル「key.txt」スクリプト設定ファイル「list.txt」「user_list.txt」を保存します。
　※list.txtの内容はuser_list.txtに追加され、list.txtは空になります。
